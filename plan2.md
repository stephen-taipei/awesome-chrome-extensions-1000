## 網頁檢測 (101-120)

### 101 - DOM Explorer

| 項目 | 內容 |
|------|------|
| **編號** | 101 |
| **名稱** | DOM Explorer |
| **分類** | 開發者工具 / 網頁檢測 |
| **狀態** | ⬜ 未開始 |

#### 功能描述
DOM 結構瀏覽器，以樹狀圖顯示頁面 DOM 結構。支援搜尋元素、複製 XPath/CSS Selector、即時高亮。

#### 介面設計
- **Popup**: 快速元素選擇器
- **Options**: 顯示設定、複製格式
- **Side Panel**: 完整 DOM 樹

#### 技術重點
- 遞迴遍歷 DOM 樹
- 虛擬化渲染大型樹狀結構
- XPath/CSS Selector 生成

#### 權限需求
| 權限 | 用途 |
|------|------|
| `activeTab` | 讀取頁面 DOM |
| `scripting` | 注入檢測腳本 |
| `sidePanel` | 樹狀檢視 |
| `storage` | 儲存設定 |
