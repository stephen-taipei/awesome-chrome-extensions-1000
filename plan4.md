### 301 - Cookie Auto-Cleaner
| 項目 | 內容 |
|------|------|
| **編號** | 301 |
| **名稱** | Cookie Auto-Cleaner |
| **分類** | 隱私與安全 / 追蹤防護 |
| **狀態** | ⬜ 未開始 |

#### 功能描述
自動清理 Cookie 工具，關閉分頁時自動刪除 Cookie，支援白名單保護重要網站。

#### 介面設計
- **Popup**: 當前網站 Cookie 數量、快速清理按鈕
- **Options**: 白名單管理、清理規則、定時清理設定
- **Content**: 無

#### 技術重點
- tabs.onRemoved 事件監聽
- Cookie API 批次操作
- 白名單比對演算法
- 清理統計記錄

#### 權限需求
| 權限 | 用途 |
|------|------|
| `cookies` | Cookie 存取和刪除 |
| `storage` | 儲存白名單設定 |
| `tabs` | 監聽分頁關閉 |
