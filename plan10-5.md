### 981 - Quantum Random Generator

| 項目 | 內容 |
|------|------|
| **編號** | 981 |
| **名稱** | Quantum Random Generator |
| **分類** | 實驗性功能 / 未來概念 |
| **狀態** | ⬜ 未開始 |

#### 功能描述
使用公開量子隨機數 API 產生真正的隨機數，用於密碼生成、抽獎等場景。

#### 介面設計
- **Popup**: 隨機數產生器、快速複製
- **Options**: 隨機數格式、來源設定
- **SidePanel**: 進階隨機數工具

#### 技術重點
- 量子隨機數 API 整合
- 多種隨機格式（數字、字串、UUID）
- 本地 CSPRNG 備援
- 隨機池快取

#### 權限需求
| 權限 | 用途 |
|------|------|
| `storage` | 快取隨機數池 |
| `clipboardWrite` | 複製隨機數 |
