### 961 - Crypto Wallet Viewer

| 項目 | 內容 |
|------|------|
| **編號** | 961 |
| **名稱** | Crypto Wallet Viewer |
| **分類** | 實驗性功能 / 區塊鏈應用 |
| **狀態** | ⬜ 未開始 |

#### 功能描述

- **Options**: 追蹤錢包管理、網路選擇
- **SidePanel**: 完整錢包資訊面板

#### 技術重點
- 公開 API 資料擷取
- 多鏈支援（ETH、BTC 等）
- 餘額即時更新
- 地址格式驗證

#### 權限需求
| 權限 | 用途 |
|------|------|
| `storage` | 儲存追蹤錢包 |
