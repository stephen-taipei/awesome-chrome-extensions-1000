### 861 - Auto Form Filler

| 項目 | 內容 |
|------|------|
| **編號** | 861 |
| **名稱** | Auto Form Filler |
| **分類** | 實用工具 / 自動化工具 |
| **狀態** | ⬜ 未開始 |

#### 功能描述
自動填寫網頁表單，支援儲存多組個人資料與自動偵測欄位。

#### 介面設計
- **Popup**: 資料組選擇、快速填寫按鈕
- **Options**: 個人資料管理、欄位對應設定
- **Content**: 表單欄位偵測與自動填入

#### 技術重點
- 表單欄位自動偵測
- 多組資料設定檔
- 欄位類型識別（name、email、phone 等）
- 安全資料儲存

#### 權限需求
| 權限 | 用途 |
|------|------|
| `activeTab` | 存取頁面表單 |
| `scripting` | 注入填寫腳本 |
| `storage` | 儲存個人資料 |
