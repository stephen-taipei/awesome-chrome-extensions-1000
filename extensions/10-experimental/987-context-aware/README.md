# Context Aware (Extension 987)

## Description
An intelligent extension that adapts your browser's behavior based on your current contextâ€”whether it's the time of day, your physical location, or your current activity.

## Features
- **Time-Based Rules**: Automatically switch themes or block sites during work hours.
- **Location Awareness**: Enable specific extensions when you're at the office or home.
- **Activity Detection**: Detects if you're reading, watching video, or coding (simulated) and adjusts settings.
- **Automation**: Triggers actions like "Do Not Disturb" based on context rules.

## Permissions
- `alarms`: To periodically check for time-based rule triggers.
- `geolocation`: To determine the user's physical location for location-based contexts.
- `storage`: To save user-defined context rules.
