# Tab Hibernator (Extension 978)

## Description
An experimental extension that automatically hibernates inactive tabs to save memory and CPU resources, with smart wake-up features and session preservation.

## Features
- **Auto-Hibernate**: Automatically suspends tabs after configurable idle time.
- **Memory Savings**: Shows real-time memory savings from hibernated tabs.
- **Whitelist**: Keep important tabs always active with domain whitelisting.
- **Quick Wake**: Instantly restore hibernated tabs with preserved scroll position.
- **Bulk Actions**: Hibernate or wake all tabs with one click.

## Permissions
- `tabs`: Required to monitor and manage tab states.
- `storage`: To save hibernation settings and whitelist.
- `alarms`: For scheduled hibernation checks.
