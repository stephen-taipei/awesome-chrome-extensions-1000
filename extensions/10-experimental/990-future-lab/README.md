# Future Lab (Extension 990)

## Description
A sandbox environment for managing and testing experimental browser features. It serves as a control center for enabling and configuring futuristic concepts before they become mainstream.

## Features
- **Feature Toggles**: Easily enable or disable experimental modules.
- **Concept Playground**: Test new ideas like Quantum RNG or Holographic UI.
- **Safety Net**: Isolated environment prevents experimental features from crashing the browser.
- **Feedback Loop**: Direct channel to report bugs or suggestions for experiments.

## Permissions
- `storage`: To save the state of enabled experiments.
