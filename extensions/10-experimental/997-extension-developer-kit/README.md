# Extension Developer Kit (Extension 997)

## Description
A utility extension designed to assist developers in creating, testing, and scaffolding new extensions for the Awesome Chrome Extensions project.

## Features
- **Manifest Generator**: Quickly generate standard Manifest V3 JSON templates.
- **README Scaffolding**: One-click Markdown templates for documentation.
- **Utility Actions**: Quick access to reload the extension or clear local storage for testing.
- **Code Snippets**: Copy-paste ready code blocks for common extension patterns.

## Permissions
- `storage`: Required to demonstrate storage clearing functionality and save dev preferences.
