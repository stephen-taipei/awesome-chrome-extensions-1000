# Password Generator (Extension 974)

## Description
An experimental extension that generates strong, customizable passwords with security analysis and memorable passphrase options for better security practices.

## Features
- **Strong Passwords**: Generate cryptographically secure random passwords.
- **Custom Rules**: Set length, character types, and exclusions.
- **Passphrase Mode**: Create memorable passphrases from word lists.
- **Strength Meter**: Visual indicator of password strength.
- **History**: Track generated passwords with encrypted local storage.

## Permissions
- `clipboardWrite`: Required to copy passwords to clipboard.
- `storage`: To save password generation preferences.
- `activeTab`: For auto-fill functionality on password fields.
