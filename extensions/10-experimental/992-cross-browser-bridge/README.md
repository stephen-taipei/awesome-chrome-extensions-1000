# Cross-Browser Bridge (Extension 992)

## Description
Facilitates the migration of extension settings and data between different browsers (e.g., Chrome to Firefox, Edge to Chrome) by using a universal JSON export format.

## Features
- **Universal Export**: Packs extension configuration into a standardized JSON file.
- **Import Wizard**: Easily restore settings from a backup file.
- **Selective Backup**: Choose to include or exclude sensitive data like encrypted passwords.
- **Cross-Platform**: Designed to work with the counterpart extension on other browsers.

## Permissions
- `storage`: Access to settings that need to be exported/imported.
- `downloads`: Required to save the exported JSON file to the user's computer.
