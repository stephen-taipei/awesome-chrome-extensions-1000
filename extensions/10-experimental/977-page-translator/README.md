# Page Translator (Extension 977)

## Description
An experimental extension that provides instant full-page translation with support for multiple translation engines, inline translation, and language learning features.

## Features
- **Full Page Translation**: Translate entire pages while preserving layout and formatting.
- **Inline Translation**: Hover over text to see instant translations without leaving the page.
- **Multiple Engines**: Choose between different translation providers for best results.
- **Language Detection**: Automatically detects source language and suggests translations.
- **Learning Mode**: Shows original text alongside translations for language learning.

## Permissions
- `activeTab`: Required to access and modify page content for translation.
- `storage`: To save language preferences and translation history.
- `contextMenus`: For right-click translation options.
