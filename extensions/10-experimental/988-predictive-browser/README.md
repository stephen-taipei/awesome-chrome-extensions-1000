# Predictive Browser (Extension 988)

## Description
A smart extension that analyzes your browsing history to predict which website you are likely to visit next, saving you time and keystrokes.

## Features
- **Smart Predictions**: Uses local algorithms to forecast your next destination.
- **Quick Access**: One-click navigation to the most probable site.
- **Probability Scores**: Shows confidence levels for each suggestion.
- **Preloading**: Optionally preloads the top hit for instant page loads.

## Permissions
- `history`: Analyzes browsing patterns to build the prediction model.
- `storage`: Saves the trained model and user preferences locally.
