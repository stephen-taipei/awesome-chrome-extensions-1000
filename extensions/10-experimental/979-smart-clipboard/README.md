# Smart Clipboard (Extension 979)

## Description
An experimental extension that supercharges your clipboard with history tracking, smart formatting, and intelligent paste suggestions based on context and content type.

## Features
- **Clipboard History**: Stores unlimited clipboard entries with search and filtering.
- **Smart Formatting**: Auto-detects content type and offers formatting options (plain text, rich text, code).
- **Context Paste**: Suggests the best clipboard item based on the current input field context.
- **Snippets Library**: Save frequently used text snippets for quick access.
- **Cross-Device Sync**: Optionally sync clipboard history across devices.

## Permissions
- `clipboardRead`: Required to access clipboard content.
- `clipboardWrite`: To paste formatted content.
- `storage`: To save clipboard history and snippets.
