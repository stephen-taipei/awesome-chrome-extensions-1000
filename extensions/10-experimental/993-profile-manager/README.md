# Profile Manager (Extension 993)

## Description
A powerful profile management tool for the Awesome Chrome Extensions project. It allows users to create and switch between different configuration sets for their extensions.

## Features
- **Multiple Profiles**: Create distinct profiles for Work, Personal, Gaming, etc.
- **Quick Switching**: Instantly toggle between profiles to change your browser environment.
- **Context Isolation**: Keep your work tools separate from your entertainment extensions.
- **Settings Inheritance**: Option to inherit global settings across profiles.

## Permissions
- `storage`: To save profile configurations and the currently active profile state.
