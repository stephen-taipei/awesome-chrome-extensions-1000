# Emotion Engine (Extension 986)

## Description
An experimental extension that attempts to detect the emotional tone of the content you are browsing and subtly adjusts the browser interface to harmonize with or improve your emotional state.

## Features
- **Sentiment Analysis**: Scans webpage text to determine if content is positive, negative, or neutral.
- **Mood Mirroring**: Visual feedback (icons, meters) reflecting the current content's tone.
- **Adaptive Response**: Switch between "Calm", "Focus", or "Cheer Up" modes to adjust themes accordingly.
- **Visual Comfort**: Automatically triggers soothing colors when high-stress content is detected.

## Permissions
- `activeTab`: Required to analyze the text content of the current tab for sentiment analysis.
- `storage`: To save user preferences for mood responses.
