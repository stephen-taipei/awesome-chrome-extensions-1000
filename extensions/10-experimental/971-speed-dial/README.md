# Speed Dial (Extension 971)

## Description
An experimental extension that replaces your new tab page with a customizable speed dial featuring your favorite sites, widgets, and a beautiful dashboard.

## Features
- **Visual Bookmarks**: Large, visual tiles for your most visited sites.
- **Custom Backgrounds**: Set personal photos or dynamic wallpapers.
- **Widgets**: Add clock, weather, todo list, and notes widgets.
- **Folder Organization**: Group speed dial entries into folders.
- **Quick Search**: Integrated search bar with multiple engine support.

## Permissions
- `storage`: To save speed dial configuration and preferences.
- `topSites`: To suggest frequently visited sites.
- `bookmarks`: To import bookmarks into speed dial.
