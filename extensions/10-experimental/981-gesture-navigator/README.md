# Gesture Navigator (Extension 981)

## Description
An experimental extension that transforms your mouse or trackpad into a gesture-based navigation system, enabling intuitive browser control through customizable drawing patterns.

## Features
- **Mouse Gestures**: Draw patterns with your mouse to trigger browser actions.
- **Rocker Gestures**: Use left-right mouse button combinations for quick navigation.
- **Gesture Trail**: Visual feedback showing your gesture path in real-time.
- **Custom Patterns**: Create and assign your own gesture patterns to any action.
- **Gesture Library**: Pre-built gestures for common actions like back, forward, refresh.

## Permissions
- `activeTab`: Required to capture mouse movements on web pages.
- `storage`: To save custom gesture configurations.
- `tabs`: To execute tab-related gesture commands.
