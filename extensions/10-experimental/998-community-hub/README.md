# Community Hub (Extension 998)

## Description
The central hub for the Awesome Chrome Extensions community. Stay updated with the latest news, access tutorials, and connect with other users.

## Features
- **News Feed**: Latest announcements and updates about the project.
- **Resources**: Quick access to tutorials, documentation, and guides.
- **Social Integration**: Links to GitHub, Discord, and other community channels.
- **Feedback System**: Direct channel to send feedback to developers.

## Permissions
- `storage`: To save user preferences and read status of news.
- `notifications`: To alert users of important announcements (future feature).
