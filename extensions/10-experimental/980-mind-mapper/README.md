# Mind Mapper (Extension 980)

## Description
An experimental extension that automatically creates visual mind maps from your browsing session, connecting related pages and concepts to help you understand and recall your research journey.

## Features
- **Auto-Mapping**: Automatically generates mind maps from your browsing history.
- **Concept Extraction**: Identifies key concepts from visited pages and creates nodes.
- **Visual Connections**: Draws relationships between related pages and topics.
- **Session Snapshots**: Save mind map snapshots at any point during research.
- **Export Options**: Export maps as images, JSON, or interactive HTML files.

## Permissions
- `history`: Required to analyze browsing patterns for mind map creation.
- `tabs`: To track current browsing session and page relationships.
- `storage`: To save mind maps and user preferences.
