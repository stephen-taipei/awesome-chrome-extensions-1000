# Tab Grouper (Extension 969)

## Description
An experimental extension that automatically organizes your tabs into smart groups based on domain, topic, or custom rules for better tab management.

## Features
- **Auto-Grouping**: Automatically group tabs by domain or category.
- **Smart Categories**: AI-powered grouping by topic (work, social, shopping, etc.).
- **Custom Rules**: Create your own grouping rules based on URL patterns.
- **Color Coding**: Assign colors to groups for easy identification.
- **One-Click Actions**: Collapse, expand, or close entire groups instantly.

## Permissions
- `tabs`: Required to access and organize browser tabs.
- `tabGroups`: To create and manage tab groups.
- `storage`: To save grouping rules and preferences.
