# Analytics Dashboard (Extension 996)

## Description
A personal dashboard to monitor and analyze the usage patterns of your installed Chrome extensions.

## Features
- **Usage Statistics**: View active extension counts and estimated memory savings.
- **Activity Charts**: Visual representation of your browser extension activity over the last 7 days.
- **Optimization Suggestions**: Smart recommendations to disable unused or resource-heavy extensions.
- **Trend Analysis**: Track performance improvements over time.

## Permissions
- `storage`: Used to store historical data and analysis results locally.
