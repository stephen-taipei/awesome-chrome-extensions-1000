# Screenshot Master (Extension 976)

## Description
An experimental extension that provides powerful screenshot capabilities including full-page capture, area selection, annotation tools, and instant sharing options.

## Features
- **Full Page Capture**: Capture entire scrolling pages as a single image.
- **Area Selection**: Select and capture specific regions of the page.
- **Annotation Tools**: Add arrows, text, highlights, and shapes to screenshots.
- **Instant Edit**: Crop, resize, and blur sensitive information.
- **Quick Share**: Share directly to cloud services or copy to clipboard.

## Permissions
- `activeTab`: Required to capture the current tab's content.
- `storage`: To save screenshot preferences and history.
- `downloads`: To save screenshots to local storage.
