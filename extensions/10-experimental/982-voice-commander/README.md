# Voice Commander (Extension 982)

## Description
An experimental extension that enables full voice control of your browser, allowing hands-free navigation, tab management, and web interaction through natural language commands.

## Features
- **Voice Navigation**: Navigate to websites, scroll pages, and click links using voice commands.
- **Tab Control**: Open, close, switch, and organize tabs with spoken instructions.
- **Custom Commands**: Create personalized voice shortcuts for frequently used actions.
- **Continuous Listening**: Option for always-on listening with wake word activation.
- **Voice Feedback**: Audio confirmation of executed commands for accessibility.

## Permissions
- `activeTab`: Required to execute voice commands on the current page.
- `tabs`: To manage browser tabs via voice commands.
- `storage`: To save custom voice commands and preferences.
