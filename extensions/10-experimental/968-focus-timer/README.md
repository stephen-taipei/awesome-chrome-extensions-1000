# Focus Timer (Extension 968)

## Description
An experimental extension that implements the Pomodoro technique with website blocking, helping you stay focused and productive during work sessions.

## Features
- **Pomodoro Timer**: Customizable work and break intervals.
- **Site Blocking**: Block distracting websites during focus sessions.
- **Progress Tracking**: Track completed pomodoros and daily productivity.
- **Break Reminders**: Gentle notifications when it's time to rest.
- **Statistics**: View your focus history and trends over time.

## Permissions
- `storage`: To save timer settings and statistics.
- `alarms`: For timer functionality and notifications.
- `notifications`: To alert when sessions end.
- `tabs`: To manage blocked sites during focus mode.
