# Note Anywhere (Extension 972)

## Description
An experimental extension that allows you to attach sticky notes to any webpage, creating persistent annotations that appear whenever you revisit the page.

## Features
- **Sticky Notes**: Create draggable notes anywhere on any webpage.
- **Rich Text**: Format notes with bold, italic, lists, and highlights.
- **Color Coding**: Organize notes with different colors and categories.
- **Auto-Save**: Notes automatically save and sync across sessions.
- **Search Notes**: Find notes across all websites with powerful search.

## Permissions
- `activeTab`: Required to inject notes into webpages.
- `storage`: To save notes and their positions.
- `tabs`: To track which page notes belong to.
