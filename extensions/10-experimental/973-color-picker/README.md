# Color Picker (Extension 973)

## Description
An experimental extension that provides a powerful color picker tool to extract colors from any webpage, manage color palettes, and convert between color formats.

## Features
- **Eyedropper Tool**: Pick any color directly from the webpage with precision.
- **Color Formats**: Convert between HEX, RGB, HSL, and CMYK formats instantly.
- **Palette Manager**: Save and organize colors into custom palettes.
- **Color History**: Track recently picked colors for quick access.
- **Gradient Generator**: Create beautiful gradients from selected colors.

## Permissions
- `activeTab`: Required to access page content for color picking.
- `storage`: To save color palettes and preferences.
- `clipboardWrite`: To copy color values to clipboard.
