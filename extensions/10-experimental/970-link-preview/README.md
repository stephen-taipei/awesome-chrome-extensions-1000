# Link Preview (Extension 970)

## Description
An experimental extension that shows instant previews of links when you hover over them, letting you peek at content before clicking.

## Features
- **Hover Preview**: See a thumbnail preview of any link by hovering over it.
- **Quick Info**: Display page title, description, and favicon instantly.
- **Preview Size**: Adjustable preview window size from small to large.
- **Keyboard Toggle**: Press a key to show/hide previews while hovering.
- **Blacklist**: Exclude specific domains from showing previews.

## Permissions
- `activeTab`: Required to inject preview functionality into pages.
- `storage`: To save preview preferences and blacklist.
