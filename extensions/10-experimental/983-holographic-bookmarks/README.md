# Holographic Bookmarks (Extension 983)

## Description
An experimental extension that transforms traditional bookmarks into rich, multi-dimensional holographic cards with live previews, spatial organization, and immersive navigation through your saved content.

## Features
- **3D Bookmark Space**: Navigate your bookmarks in a three-dimensional space organized by category, time, and relevance.
- **Live Preview Cards**: Bookmarks display as cards with live thumbnails and real-time content updates.
- **Spatial Memory**: Arrange bookmarks spatially and remember their positions for visual recall.
- **Depth Layers**: Organize bookmarks in layers - frequently used ones float to the front automatically.
- **Holographic Search**: Search through bookmarks with visual highlighting that illuminates matches in 3D space.

## Permissions
- `bookmarks`: Required to access and manage browser bookmarks.
- `storage`: To save spatial arrangements and user preferences.
- `tabs`: To generate live previews of bookmarked pages.
