# Reading Mode (Extension 975)

## Description
An experimental extension that transforms cluttered web pages into clean, distraction-free reading experiences with customizable typography and reading progress tracking.

## Features
- **Clean View**: Strips away ads, sidebars, and distractions automatically.
- **Custom Typography**: Choose fonts, sizes, line spacing, and color themes.
- **Reading Progress**: Track how far you've read with a visual progress bar.
- **Text-to-Speech**: Listen to articles with natural voice synthesis.
- **Save for Later**: Save articles for offline reading.

## Permissions
- `activeTab`: Required to modify page content for reading mode.
- `storage`: To save reading preferences and saved articles.
- `tts`: For text-to-speech functionality.
