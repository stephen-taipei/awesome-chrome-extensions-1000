# Theme Engine (Extension 994)

## Description
A centralized theme manager for the Awesome Chrome Extensions ecosystem, allowing users to customize the look and feel across all installed extensions.

## Features
- **Theme Presets**: Built-in Light, Dark, and Sepia themes.
- **Color Customization**: Fine-tune primary, accent, and background colors.
- **Global Application**: Apply theme changes to all compatible extensions with one click.
- **Live Preview**: See color changes instantly (in the popup).

## Permissions
- `storage`: To save theme preferences and share them with other extensions.
