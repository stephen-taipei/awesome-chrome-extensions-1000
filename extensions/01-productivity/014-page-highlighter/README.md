# Page Highlighter

Highlight text on any webpage and save your highlights.

## Features

- 6 highlight colors to choose from
- Save highlights per page
- View all highlights in popup
- Delete individual highlights
- Clear all highlights
- Persistent storage

## Usage

1. Select text on any webpage
2. Click the extension icon
3. Choose a highlight color
4. Click "Highlight Selection"
5. View saved highlights in the popup
6. Delete highlights as needed
