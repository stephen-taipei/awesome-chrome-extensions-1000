# Website Blocker

Block distracting websites to stay focused and productive.

## Features

- Block any website by domain
- Quick add popular social sites
- Master toggle to enable/disable
- Custom blocked page
- Favicon display for blocked sites

## Usage

1. Click the extension icon
2. Enable blocking with the toggle
3. Add websites to block
4. Use quick add buttons for popular sites
5. Remove sites by clicking Remove
