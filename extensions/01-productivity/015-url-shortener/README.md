# URL Shortener

Quickly shorten URLs and manage your shortened links.

## Features

- One-click URL shortening
- Copy shortened URL to clipboard
- History of shortened URLs
- Click history items to copy
- Uses is.gd API

## Usage

1. Click the extension icon on any page
2. Click "Shorten URL" button
3. Copy the shortened URL
4. View recent links in history
5. Click any history item to copy
