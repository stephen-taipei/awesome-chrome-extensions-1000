# Quick Notes

A simple and elegant note-taking extension for Chrome.

## Features

- Create and edit notes quickly
- Search through your notes
- Notes are saved locally
- Clean and intuitive interface
- Keyboard shortcuts support

## Usage

1. Click the extension icon to open Quick Notes
2. Click the + button to create a new note
3. Use the search bar to find notes
4. Click on a note to edit it
5. Press Ctrl+Enter to save, Escape to cancel

## Keyboard Shortcuts

- `Ctrl + Enter`: Save note
- `Escape`: Cancel editing
