# Screenshot Capture

Capture visible area screenshots with one click.

## Features

- Capture visible area instantly
- PNG and JPEG format support
- Download or copy to clipboard
- Screenshot history (last 10)
- Click history to view again

## Usage

1. Click the extension icon
2. Click "Visible Area" to capture
3. Preview the screenshot
4. Download or copy to clipboard
5. Access recent screenshots in history
