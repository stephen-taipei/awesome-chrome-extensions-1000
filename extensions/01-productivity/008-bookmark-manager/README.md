# Bookmark Manager

A clean and efficient bookmark management extension for Chrome.

## Features

- View all bookmarks in one place
- Quick search through bookmarks
- Filter by folders
- View recent bookmarks
- Add current page with one click
- Copy bookmark URLs
- Delete bookmarks
- Favicon display

## Usage

1. Click the extension icon to open Bookmark Manager
2. Use the search bar to find bookmarks
3. Click folder tabs to filter by folder
4. Click + to bookmark the current page
5. Click a bookmark to open it
6. Hover to see copy and delete options
