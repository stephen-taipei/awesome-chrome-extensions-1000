# Dark Mode Toggle

Toggle dark mode on any website with a single click.

## Features

- One-click dark mode toggle
- Adjustable brightness, contrast, and sepia
- Preset modes (Dark, Dim, Sepia)
- Settings saved per site
- Images remain visible

## Usage

1. Click the extension icon
2. Toggle dark mode on/off
3. Adjust sliders for customization
4. Use presets for quick settings
