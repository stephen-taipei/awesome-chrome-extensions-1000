# Clipboard History

Keep track of your clipboard history and quickly access past copies.

## Features

- Automatic clipboard capture on popup open
- Search through clipboard history
- Click to copy any item
- Delete individual items
- Clear all history
- Stores up to 50 items
- Relative time display

## Usage

1. Click the extension icon to open Clipboard History
2. Your current clipboard is automatically captured
3. Click any item to copy it back to clipboard
4. Use the search bar to find specific items
5. Click Delete to remove individual items
6. Click Clear to remove all history
