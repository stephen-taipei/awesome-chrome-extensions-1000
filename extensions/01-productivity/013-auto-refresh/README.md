# Auto Refresh

Automatically refresh web pages at custom intervals.

## Features

- Preset intervals (5s, 10s, 30s, 1m, 5m, 10m)
- Custom interval support
- Live countdown timer
- Per-tab refresh control
- Persists through browser restart

## Usage

1. Click the extension icon on any page
2. Select a refresh interval
3. Click Start to begin auto-refresh
4. Click Stop to disable
5. Countdown shows time until next refresh
