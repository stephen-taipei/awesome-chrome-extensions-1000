# Tab Counter

Display the number of open tabs with detailed statistics.

## Features

- Total tab count
- Window, pinned, and audio tab counts
- Top domains breakdown
- Close duplicate tabs
- Estimated memory usage
- Real-time updates

## Usage

1. Click the extension icon
2. View your tab statistics
3. See top domains with most tabs
4. Click "Close Duplicate Tabs" to clean up
5. Monitor estimated memory usage
