# Task Timer

A Pomodoro-style productivity timer for focused work sessions.

## Features

- Pomodoro technique with 25-minute work sessions
- Short (5 min) and long (15 min) break timers
- Visual progress circle
- Task tracking
- Session and total time statistics
- Desktop notifications
- Automatic mode switching

## Usage

1. Click the extension icon to open Task Timer
2. Enter what you're working on (optional)
3. Click Start to begin a work session
4. Take breaks when prompted
5. Track your productivity stats

## The Pomodoro Technique

1. Work for 25 minutes
2. Take a 5-minute break
3. After 4 work sessions, take a 15-minute break
4. Repeat
