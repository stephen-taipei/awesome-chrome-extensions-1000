# Session Manager

Save and restore browser tab sessions with one click.

## Features

- Save all current tabs as a session
- Name your sessions for easy identification
- View tabs preview for each session
- Restore sessions with one click
- Delete old sessions
- Date tracking for sessions

## Usage

1. Click the extension icon to open Session Manager
2. View your current tabs at the top
3. Click Save to save the current session
4. Enter a name for your session
5. Click Restore on any saved session to reopen those tabs
6. Delete sessions you no longer need
