# QR Code Generator

Generate QR codes for any URL or text instantly.

## Features

- Generate QR for current page URL
- Custom text/URL input
- Multiple sizes (S, M, L)
- Download as PNG
- Copy to clipboard
- Real-time preview

## Usage

1. Click the extension icon
2. QR code for current URL is generated
3. Switch to Custom tab for custom text
4. Choose QR code size
5. Download or copy the QR code
