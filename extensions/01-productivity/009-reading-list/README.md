# Reading List

Save articles and pages to read later with progress tracking.

## Features

- Save any page to your reading list
- Mark articles as read/unread
- Separate tabs for unread and read articles
- Favicon display for easy identification
- Delete articles when done
- Relative date display

## Usage

1. Click the extension icon on any page
2. Click + to add the current page
3. View your saved articles
4. Click Open to read an article
5. Mark as read when finished
6. Switch tabs to see read articles
