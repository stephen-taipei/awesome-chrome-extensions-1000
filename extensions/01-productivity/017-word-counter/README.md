# Word Counter

Count words, characters, sentences, and paragraphs in selected text or entire page.

## Features

- Count selected text or entire page
- Word count
- Character count (with and without spaces)
- Sentence count
- Paragraph count
- Estimated reading time
- Text preview

## Usage

1. Select text on any webpage (optional)
2. Click the extension icon
3. View stats for selection or entire page
4. Toggle between Selection and Entire Page
5. Click Refresh to update counts
