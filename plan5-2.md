## 音樂播放工具（續）

### 421 - Scrobbler
| 項目 | 內容 |
|------|------|
| **編號** | 421 |
| **名稱** | Scrobbler |
| **分類** | 娛樂 / 音樂播放 |
| **狀態** | ⬜ 未開始 |

#### 功能描述
音樂記錄工具，自動記錄播放歷史到 Last.fm 或其他服務、追蹤收聽統計。

#### 介面設計
- **Popup**: 最近記錄、統計摘要
- **Options**: 服務連接、記錄條件設定、支援平台
- **Content**: 無

#### 技術重點
- 各音樂平台播放偵測
- Last.fm API 整合
- Scrobble 條件判斷（播放時長）
- 離線佇列處理

#### 權限需求
| 權限 | 用途 |
|------|------|
| `storage` | 儲存記錄佇列 |
| `host_permissions` | 音樂平台和 API 存取 |
