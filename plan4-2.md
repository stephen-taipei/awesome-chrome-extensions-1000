## 密碼管理工具（續）

### 321 - Secure Notes Vault
| 項目 | 內容 |
|------|------|
| **編號** | 321 |
| **名稱** | Secure Notes Vault |
| **分類** | 隱私與安全 / 密碼管理 |
| **狀態** | ⬜ 未開始 |

#### 功能描述
加密筆記保管庫，使用主密碼加密儲存敏感筆記、API 金鑰、軟體序號等。

#### 介面設計
- **Popup**: 快速檢視、搜尋筆記、新增按鈕
- **Options**: 主密碼設定、加密選項、自動鎖定
- **SidePanel**: 完整筆記管理介面

#### 技術重點
- Web Crypto API 加密（AES-GCM）
- 主密碼衍生金鑰（PBKDF2）
- 自動鎖定計時器
- 安全剪貼簿複製

#### 權限需求
| 權限 | 用途 |
|------|------|
| `storage` | 儲存加密資料 |
| `sidePanel` | 筆記管理面板 |
| `clipboardWrite` | 安全複製 |
