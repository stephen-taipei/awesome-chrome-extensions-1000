## 剪貼簿管理 (061-070)

### 061 - Clipboard History

| 項目 | 內容 |
|------|------|
| **編號** | 061 |
| **名稱** | Clipboard History |
| **分類** | 生產力工具 / 剪貼簿管理 |
| **狀態** | ⬜ 未開始 |

#### 功能描述
剪貼簿歷史記錄，自動儲存複製內容。支援文字、圖片、連結分類。一鍵貼上歷史項目。

#### 介面設計
- **Popup**: 歷史列表、搜尋、快速貼上
- **Options**: 保留數量、自動清理、隱私排除
- **Content**: 無

#### 技術重點
- 監聽 clipboard API
- 圖片轉 Base64 儲存
- 敏感內容自動排除

#### 權限需求
| 權限 | 用途 |
|------|------|
| `clipboardRead` | 讀取剪貼簿 |
| `clipboardWrite` | 寫入剪貼簿 |
| `storage` | 儲存歷史 |
