### 601 - Price History Tracker
| 項目 | 內容 |
|------|------|
| **編號** | 601 |
| **名稱** | Price History Tracker |
| **分類** | 購物與財務 / 比價購物 |
| **狀態** | ⬜ 未開始 |

#### 功能描述
價格歷史追蹤器，記錄商品價格變化並以圖表顯示趨勢。

#### 介面設計
- **Popup**: 當前商品價格、歷史最低價
- **Options**: 追蹤設定、通知偏好
- **Content**: 商品頁面價格圖表注入

#### 技術重點
- 商品資訊自動擷取
- IndexedDB 價格歷史儲存
- Chart.js 圖表視覺化
- 價格變化通知

#### 權限需求
| 權限 | 用途 |
|------|------|
| `storage` | 儲存價格歷史 |
| `activeTab` | 擷取商品資訊 |
| `notifications` | 價格變化提醒 |
| `alarms` | 定期檢查價格 |
