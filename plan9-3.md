### 841 - PDF Quick Viewer

| 項目 | 內容 |
|------|------|
| **編號** | 841 |
| **名稱** | PDF Quick Viewer |
| **分類** | 實用工具 / 文件處理 |
| **狀態** | ⬜ 未開始 |

- **Popup**: PDF 預覽視窗、縮放控制、頁面導航

- **Content**: 攔截 PDF 連結提供預覽選項

- PDF.js 整合
- Canvas 渲染
- 頁面快取機制
- 縮放與滾動同步

#### 權限需求
| 權限 | 用途 |
|------|------|
| `activeTab` | 存取當前頁面 PDF 連結 |
| `storage` | 儲存使用者設定 |
